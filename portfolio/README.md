# 👩‍💻 Portfólio de Tatiane Ramos

Aplicação web desenvolvida para apresentação profissional, destacando experiência como Desenvolvedora Full Stack, projetos e stack tecnológica.

🔗 [Acesse o portfólio online](https://tatiane2022.github.io/portfolio)

---

## 🚀 Sobre o projeto
Este portfólio foi desenvolvido com foco em organização, clareza e responsividade, permitindo apresentar projetos, habilidades técnicas e experiência profissional de forma objetiva.

---

## 🛠️ Tecnologias utilizadas
- HTML5, CSS3 e JavaScript
- Angular (TypeScript)
- C# e .NET
- SQL Server

---

## 💡 Funcionalidades
- Estrutura responsiva para diferentes dispositivos
- Organização de seções (Sobre, Tecnologias, Projetos e Contato)
- Navegação simples e intuitiva
- Integração com GitHub e LinkedIn

---

## 🎯 Objetivo
Centralizar informações profissionais, projetos e tecnologias utilizadas, facilitando a visualização do meu perfil técnico e evolução como desenvolvedora.
